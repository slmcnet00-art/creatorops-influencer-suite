CreatorOps Influencer Suite - YouTube API Evidence

Prepared for YouTube Data API quota extension review. Last updated: July 3, 2026.

1. Product Overview

CreatorOps Influencer Suite is a web-based influencer campaign management platform for brands and agencies. The platform helps users discover public creators, review public channel and video performance, manage outreach, track campaign content, and generate reports.

Application URL: https://creatorops-influencer-suite.onrender.com
Privacy Policy: https://creatorops-influencer-suite.onrender.com/privacy
Terms of Service: https://creatorops-influencer-suite.onrender.com/terms

2. YouTube API Use Cases

  • Search public YouTube channels related to a brand campaign brief.
  • Retrieve public channel statistics such as subscriber count, total views, video count, and country when available.
  • Search public videos as content references for campaign planning.
  • Retrieve public video statistics such as views, likes, comments, and publish date.
  • Help brand teams compare creators and content references before outreach.
  • Track public campaign content performance after creators upload collaboration content.

3. Endpoints Used

  • youtube.search.list - public channel and video discovery.
  • youtube.channels.list - public channel statistics and metadata.
  • youtube.videos.list - public video statistics and metadata.

The service does not use youtube.videos.insert, does not upload videos, and does not modify or delete YouTube content.

4. OAuth and User Login

Users log in to CreatorOps Influencer Suite to manage their own brand campaigns, creator discovery lists, outreach workflow, and campaign reports. YouTube Data API access is mainly used to retrieve public YouTube channel and video data. We do not require users to grant access to private YouTube account data for creator discovery.

Google OAuth may be used for Google Workspace features such as Gmail sending and Google Sheets export. These are separate from public YouTube discovery and are used only with user authorization.

5. Data Handling

  • We only use publicly available YouTube channel and video data.
  • We do not collect private YouTube user data.
  • We do not store YouTube OAuth tokens for private YouTube account access.
  • We do not allow users to download or redistribute YouTube videos.
  • We store public metadata such as channel URLs, video URLs, subscriber counts, view counts, like counts, comment counts, publish dates, and campaign notes.
  • We cache repeated search results to reduce unnecessary API calls and quota usage.

6. User Flow

  1. A brand or agency user logs in to CreatorOps Influencer Suite.
  2. The user creates a campaign and enters brand, product, target persona, keywords, campaign schedule, KPI, and target creator count.
  3. The user opens the Discovery page and searches for creators by keyword, platform, category, and country.
  4. The platform uses YouTube Data API endpoints to find public YouTube channels/videos and retrieve public metrics.
  5. The user reviews candidates, saves selected creators to a pre-message candidate pool, and prepares outreach.
  6. After campaign content is uploaded, the user registers public content URLs and refreshes public performance metrics for reporting.

7. Requested Quota

  • Total daily quota: 100,000 units per day.
  • Maximum per-minute quota: 600 units per minute.
  • Search Queries per day: 10,000 initially, preferably 50,000 per day.

CreatorOps Influencer Suite will comply with the YouTube API Services Terms of Service, Google Privacy Policy requirements, and applicable developer policies. Users can access the privacy policy and terms from the application interface.
