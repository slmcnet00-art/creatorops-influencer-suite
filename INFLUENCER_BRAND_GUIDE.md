## 2. 캠페인 목적

예시:

- 제품 사용법 인지

## 4. 원메시지

브랜드 또는 제품이 영상에서 반드시 전달해야 하는 한 문장이다. 캠페인 후킹 포인트와 제안 메시지의 중심 문구로 사용한다.

작성 예시:
- “좁은 공간에서도 반려견이 편안하게 이동할 수 있는 항공 규격 이동식 켄넬”
- “민감 피부도 매일 쓰기 편한 저자극 장벽 세럼”
- “초보 셀러도 짧은 영상으로 구매 전환을 만들 수 있는 공동구매 상품”

## 5. 타깃 페르소나

영상이 공감시켜야 할 고객 특성을 구체적으로 작성한다.

필수 입력:
- 연령대
- 성별 또는 라이프스타일
- 주요 고민
- 구매를 망설이는 이유
- 구매를 결심하는 조건

예시:
- “30대 초반, 혼자 자취하며 반려견과 자주 이동하는 직장인”
- “20대 중반, 10년째 피부 트러블 고민이 있어 성분과 후기를 꼼꼼히 보는 여성”

예시:

- 실제 사용 시나리오가 명확함
- 고객 불안 요소를 해소함

## 7. 영상 제작 방향

원하는 영상 유형을 구체적으로 선택한다.

가능한 유형:
- 상세 리뷰
- 감성 브랜딩
- 인터뷰
- 룩북
- 언박싱
- 사용법 시연
- 비교 리뷰
- 문제 제기 후 해결
- 감각 자극형 숏폼
- 공동구매/판매 유도형

## 8. 벤치마킹 레퍼런스

타 브랜드 영상이나 원하는 톤의 영상을 링크로 기록한다.

| 링크 | 참고할 점 | 따라 하면 안 되는 점 |
| --- | --- | --- |
|  |  |  |

주의:
- 레퍼런스는 방향 참고용이다.
- 동일한 촬영 환경, 모델, 편집 퀄리티를 보장하지 않는다.
- 저작권 침해나 타 브랜드 실명 비교는 피한다.

## 9. 영상 내러티브

영상의 흐름을 사전에 정한다.

권장 구조:
1. 첫 3초 안에 문제, 가격, 장면, 감정 중 하나로 후킹한다.
2. 타깃 페르소나가 공감할 상황을 보여준다.
3. 제품의 핵심 기능 또는 셀링 포인트를 실제 사용 장면으로 보여준다.
4. 구매 전 망설임을 해소한다.
5. CTA 또는 다음 행동을 자연스럽게 유도한다.

예시 구조:
- 초반 명확한 문제 제기 → 제품 사용 → 문제 해소
- 궁금증 유발 → 실사용 장면 → 마지막에 제품 노출
- 가격/혜택 제시 → 사용 시나리오 → 구매 조건 안내

## 10. 자막, 나레이션, 배경음악

| 항목 | 가이드 |
| --- | --- |
| 자막 | 모바일에서 잘 읽히는 크기, 핵심 문장 중심 |
| 나레이션 | 과장보다 실제 사용 경험처럼 자연스럽게 |
| 배경음악 | 상업적 이용 가능한 음원만 사용 |
| 폰트/이미지 | 저작권 및 상업적 사용 가능 여부 확인 |

## 11. 크리에이터 전달 사항

크리에이터에게 반드시 전달해야 할 운영 기준이다.

- 영상은 기본적으로 모바일 시청 환경에 맞춘 세로형 9:16 비율로 제작한다.
- 별도 요청이 없으면 정방형, 가로형 영상은 제작하지 않는다.
- 첫 장면은 3초 이내 후킹 포인트가 살아야 한다.
- 영상 길이는 가능하면 30초 이내로 제작한다.
- 정의된 페르소나에 맞춰 공감대를 끌어내는 스크립트와 장면을 구성한다.
- 폰트, 음원, 이미지 사용 시 상업적 이용 가능 여부를 확인한다.
- 스튜디오 촬영이 아닌 크리에이터의 실제 촬영 환경을 전제로 한다.
- 레퍼런스 영상은 참고용이며 동일한 결과물을 보장하지 않는다.

## 12. 클라이언트 확인 사항

- 영상 제작 이후 재촬영을 제외한 자막, 효과, 컷편집 수정은 1회 가능하다.
- 최종 납품 후 수정은 불가하다.
- 정방형 또는 가로형 영상이 필요한 경우 촬영 전 요청해야 하며 추가 비용이 발생할 수 있다.
- 크리에이터의 촬영 환경상 스튜디오 촬영 형태의 완성도를 전제로 하면 안 된다.

## 13. CreatorOps 반영 방식

| CreatorOps 영역 | 반영 내용 |
| --- | --- |
| 브랜드/제품 학습자료 | 원메시지, USP, 금지 표현, 레퍼런스, 페르소나 |
| 발굴 조건 | 타깃 페르소나, 플랫폼, 카테고리, 팔로워/조회수 조건 |
| 캠페인 생성 | 캠페인 목적, 영상 유형, 미션, 리워드, 검수 플로우 |
| 제안 메시지 | 원메시지, 협업 목적, 답변 유도 문장 |
| 콘텐츠 추적 | URL, 조회수, 댓글, 공유, 저장, 전환 |
| 리포트 | KPI 목표 대비 달성률, 콘텐츠별 성과 |

## 14. 검수 체크리스트

- [ ] 원메시지가 첫 3초 안에 드러나는가?
- [ ] 타깃 페르소나가 공감할 상황이 있는가?
- [ ] USP가 말이 아니라 장면으로 보여지는가?
- [ ] 과장 표현, 의학적 효능 단정, 경쟁사 실명 비교가 없는가?
- [ ] 상업적 이용 가능한 폰트/음원/이미지를 사용했는가?
- [ ] 영상 길이가 30초 내외인가?
- [ ] CTA 또는 다음 행동이 자연스럽게 연결되는가?
- [ ] 브랜드가 원하는 톤과 크리에이터의 자연스러움이 충돌하지 않는가?
